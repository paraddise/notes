Help

Device Status page provides you with the current status of the device and the list of channels.

All information is read-only and cannot be modified.

Field Device Status displays the current status of the device (Active or Inactive).

Field Receiver Enabled shows whether the receiver is enabled or not. Receiver may be enabled only in certain software versions.

Field Channels Available shows the number of channels available for viewing.

Field IP Address shows the IP address of the device. It is now assigned only by DHCP, static configuration will be available in future versions.

Fields Subnet Mask and Internet Gateway show current internet settings of the device.

Fields Channel Frequency N show the frequency of available channels. If the field is marked as N/A, the channel is not available.

System Upgrade page allows you to upgrade the device software to the latest version.

You should download the upgrade file from the manufacturer's website and upload it to the device.

To upgrade this device's firmware, follow these instructions:

1. Download the latest firmware from the manufacturer's website.

2. Click the "Browse" button and select the firmware file.

3. Click the "Install" button to start the upgrade process.

4. The device needs to be restarted after the upgrade is complete.

Field Firmware version shows the current version of the firmware.

Field Hardware version shows the current version of the hardware. The firmware version should be compatible with the hardware version.

Note: Do not turn off the device during the upgrade process. The upgrade process may take several minutes. Loss of power during the upgrade process may damage the device.
